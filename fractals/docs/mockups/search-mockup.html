Interaction Reference

Click category Scope tag appears in search bar. Search text preserved.
Click × on scope tag Category deselected. Search text preserved. Results widen to global.
Click × clear button Search text cleared. Category scope preserved. Back to browse within category.
Escape (1st press) Clear search text (if any). Category stays.
Escape (2nd press) Clear category scope. Fully reset to browse.
Backspace (empty input) Remove category scope tag (like clicking its ×).
Click type tab Clears both search text and category. Fresh start.
/ or ⌘K Focus search bar. Scope tag stays if present.
Sidebar "Filter categories" Filters category list names only. Not a content search. Unchanged.
